Gaussq

PURPOSE

returns the locations y_i and weights w_i for Gauss quadrature of order n

SYNOPSIS

function [yi,wi] = Gaussq (n)

DESCRIPTION 

 returns the locations y_i and weights w_i for Gauss quadrature of order n
